# raykrueger/claude-plugins

Personal Claude Code plugin marketplace.

## Plugins

| Plugin | Description |
|--------|-------------|
| [discord-notifications](plugins/discord-notifications) | Send Discord notifications from Claude Code |

## Install

Add this marketplace:
```
/plugin add-marketplace raykrueger/claude-plugins
```

Then install a plugin:
```
/plugin install discord-notifications@raykrueger
```
